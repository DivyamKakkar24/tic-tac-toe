|tic-tac-toe|

A traditional tic-tac-toe game made in python (Computer vs. Player).

Rules:
-> Play occurs on a 3 by 3 grid of 9 empty squares.
-> Squares are marked alternately. First Xs and then Os are marked.
-> If a player places three of the same marks in a row, that player wins.
-> If all the blocks are filled and there is no winner, the game ends in a draw.
